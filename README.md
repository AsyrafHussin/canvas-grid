# Canvas Grid

## Getting Started

```bash
# Clone the repo
git clone https://github.com/AsyrafHussin/canvas-grid.git

# Navigate to project folder
cd canvas-grid

# Initial Setup
npm install

# Serve the application on http://192.168.58.97:9966/
npx canvas-sketch src/index.js
```
